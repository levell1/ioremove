# 2023-02-25 GitBlog 시작(MarkDwon 사용법)

MarkDown 사용법
==============

# 1. 헤더Header


* ## 1-1제목

        This is Header
        ===============
    This is Header
    ===============

* ## 1-2 작은제목

        This is H2
        ----------
    This is H2
    ----------

* ## 1-3 글머리 
    * 1~6 까지 가능

            # This is H1
            ## This is H2
            ### This is H3
            #### This is H4
            ##### This is H5
            ###### This is H6
        # This is H1
        ## This is H2
        ### This is H3
        #### This is H4
        ##### This is H5
        ###### This is H6

# 2. `&nbsp;` 띄어쓰기
* `&nbsp;`를 이용하여 띄어쓰기를 할 수 있다.
* `<br>` 줄바꿈.

        안녕하세요.<br>
            감사합니다.<br>
        &nbsp;&nbsp;어서오세요.

    안녕하세요.<br>
        감사합니다.<br>
    &nbsp;&nbsp;어서오세요.

# 3. BlockQuote

*   `>` 를 이용한다 

        >   #   First blockqute.
        >	>   ##  Second blockqute.
        >	>	>   *   ##  Third blockqute.
        >	>	>   >   *   ##  Fourth blockqute.
* 적용
    >   ##  First blockqute.
    >	>   ###  Second blockqute.
    >	>	>   *   ###  Third blockqute.
    >	>	>   >       Fourth blockqute.


# 4. 목록

* ## 3-1 순서있는 목록
    ####  &nbsp;순서있는 목록은 숫자와 점을 사용한다.

        1. First
        2. Second
        3. Third
* 적용
    1. First
    2. Second
    3. Third

* ## 3-2 순서없는 목록
    #### &nbsp; 기호 :&nbsp; * , + , &nbsp;-  를 사용한다. 기호를 혼합해서 사용 가능하다.
        * First
        * Second
        * Third     <br><br>
        
        + First
        + Second
        - Third     <br><br>

        + First
            + Second
                + Third <br><br>
        
        * First
            + Second
                - Third
    * First
    * Second
    * Third     <br><br>
    + First
    + Second
    - Third     <br><br>
    * First
        * Second
           * Third  <br><br>
    * First
        + Second
            - Third



# 5. 코드
* ## 들여쓰기
    ####  &nbsp;코드내용(한번의탭, 네개의 공백으로 만나면 변환되기 시작하여 들여쓰지 않은 행을 만날때까지 변환이 계속된다.)

        Code,Blog,C#,Unity,git

            <TAB Code Block
        End Code block
* 적용<br>
    Code,Blog,C#,Unity,git

        <TAB Code Block
    End Code block  

* ## 코드,기호를 이용
*  ### `<pre><code> Code </code></pre>`이용

        <pre>
        System.out.println("1.Learn Unity");
        </pre>
        <code>
        System.out.println("2.Learn Unity");
        </code>
        <pre><code>
        <br><br>
        System.out.println("3.Learn Unity");
        </code></pre>
    
    <pre>
    System.out.println("1.Learn Unity");
    </pre>

    <code>
    System.out.println("2.Learn Unity");
    </code><br><br>
    
    <pre><code>
    System.out.println("3.Learn Unity");
    </code></pre>
